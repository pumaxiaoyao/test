系统参数设置
以下系统参数设置关系到系统是否正常运行，请不要随意操作，确保在操作之前已经了解各个参数的作用。

    1、站点参数

站点名称
如果关闭后台登录后，登录时接收到的提示信息
默认客服在登录操作中如果连续5次输入密码错误，则锁定该客服帐号不允许登录，只能通过其他管理员客服进行解锁
前台是否显示流水检查界面（默认以及关闭为不显示，打开后显示）
用于提醒客服活动的图片分辨率
消息提示音(默认为：/incloud/newtask.wav)
第一次访问首页是否跳转到注册页(默认不跳转)

    2、系统存款设置

允许玩家未处理的存款条数
三方存款自动到账开关
如果关闭，则所有玩家将无法提交存款请求，默认打开

如果关闭，则所有玩家将无法提交网银方式的存款请求，默认打开
如果关闭，则所有玩家将无法提交三方在线支付的存款请求，默认打开
玩家使用ATM存款时，单日存款最高次数，超过后玩家当日将无法使用这种存款方式
玩家使用ATM存款时，单次存款最高限额
玩家使用ATM存款时，单次存款最小限额
玩家使用互联网存款时，单次存款最高限额
玩家使用互联网存款时，单次存款最小限额
玩家使用网银存款时，单日存款最高次数，超过后玩家当日将无法使用这种存款方式
玩家使用网银存款时，单次存款最高限额
玩家使用网银存款时，单次存款最小限额
玩家使用三方在线存款时，单日存款最高次数，超过后玩家当日将无法使用这种存款方式

玩家使用三方在线存款时，单次存款最小限额

玩家使用公司入款（网银）显示名称
玩家使用公司入款（ATM）显示名称
三方存款默认优惠比例。填写百分比数值，例如优惠百分之二，填写数值2，最高不超过5。

    3、系统取款设置

如果关闭，则所有玩家将无法提交取款请求，默认打开
超过该次数，玩家当日无法再执行取款操作，默认=5
玩家当日取款的累计值如果超过这个金额，将无法完成取款申请
玩家当次取款金额如果超过这个金额，将无法完成取款申请
玩家当次取款金额如果小于这个金额，将无法完成取款申请
当玩家取款额多于存款额指定倍数时，系统会于流水检查时提示警告信息 (默认设置是2倍)
默认为“关闭”，如果开启取款密码，个人中心开放取款密码设置，玩家在注册和取款时必须输入取款密码；未设置取款密码，取款时则无须输入
超过该次数，代理当日无法再执行取款操作，默认=5
代理当日取款的累计值如果超过这个金额，将无法完成取款申请
代理当次取款金额如果超过这个金额，将无法完成取款申请
代理当次取款金额如果小于这个金额，将无法完成取款申请
默认允许，关闭则玩家、代理只能申请取款数为整数
默认为“关闭”，如果开启取款分配模式，取款将可以分配给指定的专员来进行审核与出款

    4、玩家注册登录设置

关闭则不允许所有玩家登录到系统
关闭则不允许所有新玩家注册到系统
默认玩家在登录操作中如果连续5次输入密码错误，则锁定该玩家帐号不允许登录，只能通过客服进行解锁
默认玩家注册时填写用户帐号名称不能填写为admin
如果关闭玩家登录后，玩家登录时接收到的提示信息
如果关闭玩家注册后，玩家注册时接收到的提示信息
注册时需要填写生日（默认为关闭,打开后为必填）
注册时需要填写QQ号码（默认为关闭,打开后为必填）
注册时需要填写手机号码（默认为关闭, 打开后为必填）
注册时需要填写电子邮箱（默认为关闭, 打开后为必填）
注册时检查真实姓名同名(默认为关闭，打开后限制同真实姓名注册)
是否允许玩家前台编辑个人资料（默认为关闭,打开后为可编辑。如果个人资料为注册要求必填项目，则不能保存为空。）

    5、代理注册登录设置

关闭则不允许所有推广代理用户登录到系统
关闭则不允许任何新的推广代理用户进行注册
默认推广代理用户在登录操作中如果连续5次输入密码错误，则锁定该代理帐号不允许登录，只能通过客服进行解锁
默认推广代理用户帐号名称不能填写为admin
如果关闭代理登录后，代理登录时接收到的提示信息
如果关闭代理注册后，代理登录时接收到的提示信息
代理注册时需要填写手机号码（默认为关闭,打开后为必填）
代理注册时需要填写QQ号码（默认为关闭,打开后为必填）
代理注册时需要填写邮箱（默认为关闭,打开后为必填）
