首先，IE6或其他低版本浏览器效率极低，很多比较新颖和动画效果都无法实现加载，导致用户阅读和游戏时效果极差；

其次，低版本浏览器安全性较低，黑客很容易利用其进行病毒或木马植入，从而导致你的投注体验和资金受到极大威胁。

因此，我们强烈建议您将浏览器升级到最新版本，以此来获得BEPLAY带来最好的游戏体验和优质服务。

3. 我是否需要更改任何浏览器配置？

为了访问我们网站的全部功能，您需要启用JavaScript和在您的浏览器设置接受Cookies。如果您已经禁用JavaScript和cookies，将会无法进行投注或受到严重限制。如果您的浏览器使用浏览器的默认设置，您应该是可以正常进行投注。

Cookies是当你浏览某网站时，由网站服务器置于你硬盘上的一个非常小的文本文件，它可以记录你的网站设置、投注选项和浏览过的网页等信息。Cookies记录的只是关于您浏览器中的信息，并不是关于您个人的信息。

JavaScript使网站更具互动性的一种网页浏览器编程语言，几乎所有浏览器都支持。启用JavaScrip是相当安全的，他只是在您的浏览器上运行，不会在您的电脑操作系统上运行。

然而在某些情况下，调整您浏览器的隐私或安全级别过高也会影响您的请求。这理是浏览器如何启用JavaScript和接受Cookies的方法：

启用JavaScript：

微软IE:

1) 在浏览器的任务菜单中选择“工具”然后选择“Internet选项”再选择“安全” 标签。 2) 点击“Internet”图标 3) 点击“默认级别”然后点击“确定”进行关闭“Internet选项”页面。

火狐Firefox:

1) 在浏览器的任务菜单中选择“工具”然后选择“选项”再选择“内容”。 2) 在“启用JavaScript”选项中打勾。 3) 点击“确定”进行关闭选项页面。

苹果Safari:

1) 选择Safari工具菜单中的“偏好设置”，然后选择“安全”标签。 2) 在“启用JavaScript”选项中打勾。 3) 点击窗口右上方的“X”按钮设置会默认应用。

谷歌Chrome:

1) 选择Chrome工具菜单中的“选项”。 2) 选择“高级选项”标签。 3) 在“隐私权”区域点击“内容设置”。 4) 选择“JAVASCRIPT”标签。 5) 点击选择“允许所有网站运行JAVASCRIPT（推荐）” 6) 点击“关闭”进行应用您的设置。

接受Cookies:

微软IE：

1) 在浏览器的任务菜单中选择“工具”然后选择“Internet选项”。 2) 选择“隐私”标签。 3) 将安全系数滚动条设置为中、低或接受所有Cookies。 4) 点击“确定”进行关闭“Internet选项”页面。

火狐Firefox:

1) 在浏览器的任务菜单中选择“工具”然后选择“选项”。 2) 选择“隐私”标签。 3) FIREFOX将会：“使用自定义历史记录设置” 4) 在“接受站点的COOKIE”选项中打勾。 5) 点击“确定”进行关闭选项页面。

Safari:

1) 选择Safari工具菜单中的“偏好设置”。 2) 选择“安全”标签。 3) 在“接受cookies”选项中选择“总是”或“仅从您浏览的站点”。 4) 点击窗口右上方的“X”按钮设置会默认应用。

谷歌Chrome:

1) 选择Chrome工具菜单中的“选项”。 2) 选择“高级选项”标签。 3) 在“隐私权”区域点击“内容设置”。 4) 选择“Cookies”标签。 5) 确保您已经选择了“允许设置本地数据（推荐）”。 6) 点击“关闭”进行应用您的设置。

4. 什么是推荐的显示分辨率？

对于一个良好的视觉体验，我们建议至少为1024x768像素的图像分辨率。同样我们建议您使用最低的32位颜色深度。
